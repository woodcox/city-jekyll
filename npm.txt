Node Package Manager
====================
# Altering this file causes the npm workflow to run

Create package.json - run: npm init -y

Install packages - run: npm i -D @fullhuman/postcss-purgecss postcss postcss-cli autoprefixer cssnano purgecss

Installed:
  - postcss
  - postcss-cli
  - autoprefixer
  - cssnano
  - purgecss (@fullhuman/postcss-purgecss)
